# 北邮升学就业信息平台

一个专属于北邮人的社交平台，参考小红书的界面和功能设计，专注于北京本地高校/企业内部信息分享。

## 项目功能

- **可隐藏身份的实名认证**：保证信息真实性，真实信息仅做验证不公开
- **发布信息**：用户可发布导师/企业的内部信息分享帖，包含分类、评分等信息
- **分类浏览**：用户可按分类（导师信息/企业信息）浏览分享帖
- **用户认证**：完整的注册、登录功能
- **数据存储**：使用Supabase云数据库存储数据

## 技术栈

- React 18
- React Router v6
- Vite
- Supabase (云数据库 + 认证)
- GitHub Pages (部署)

## 核心功能

### 1. 实名认证
- 用户输入真实姓名、学号、学校信息
- 信息仅用于验证，不会公开显示

### 2. 发布信息
- 选择分类（导师信息/企业信息）
- 输入目标对象名称
- 填写分享内容
- 给出综合评分

### 3. 分类浏览
- 支持按全部、导师信息、企业信息分类浏览
- 展示帖子的分类、目标对象、内容、评分、发布时间等信息

## 数据模型

### 用户表
- 用户ID
- 实名认证验证状态
- 公开昵称

### 分享帖子表
- 帖子ID
- 内容分类（导师信息/企业信息）
- 目标对象名称
- 分享内容
- 综合评分
- 发布时间
